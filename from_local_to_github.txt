* Create folder in local machine: make sure this is the location you want to
  store locally

* Add files to your current local folder

* Create the same folder in github.com
    - Upper right hand corner
    - Click on the "plus" icon
    - Choose New Repository

* Initialize git commands:
    - git init
    - git add -A
    - git commit -m "first commit"
    - git branch -M main

      # substitute valtix/gitNotes.git with the appropriate name you used for
      your new repo:
    - git remote add origin git@github.com:valtix/gitNotes.git
    - git push -u origin main
